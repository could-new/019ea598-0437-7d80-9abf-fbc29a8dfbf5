# Diversity in the Living World

A Flutter application featuring a beautifully designed, interactive presentation on the Grade 6 biology lesson "Diversity in the Living World".

## Features
- **Interactive Slides**: Swipe through slides covering key concepts like Classification, Kingdoms of Life, and Characteristics of Living Things.
- **Responsive Layout**: Designed to look great on mobile devices, tablets, and desktops.
- **Engaging Visuals**: Clean and modern presentation UI built entirely with Flutter.
- **Easy Navigation**: Built-in progress tracking and navigation controls.

## Tech Stack
- Framework: Flutter
- Language: Dart
- State Management: StatefulWidgets

## Setup & Run
1. Ensure Flutter is installed and set up on your machine.
2. Clone this repository.
3. Run `flutter pub get` to install dependencies.
4. Run `flutter run` to launch the application.
